##认识

在当今信息量剧增的世界环境中，每个人都需要从各种各样的信息源中收集和利用信息。同时，随着信息的多媒体化，信息质量的不确定性和数量的爆炸增长使我们要认识到信息素养的重要性。信息素养涉及各方面的知识,是一个特殊的、涵盖面很宽的能力，它包含诸多因素，和许多学科有着紧密的联系。信息素养，可以帮助同学们更好的利用网络信息，它是网络时代素质教育的有机内容。所以，大学生信息素养的培养刻不容缓，对于信息时代的大学生来说必须具备信息素养才可以适应社会的发展变化，信息素养是促进他们自身综合能力同社会的要求结合一起的重要素质和能力。信息素养的培养应该从信息搜集、处理和运用能力入手，培养知识创新的本领，从而实现信息向知识转化知识向能力转化、能力向技术转化的飞跃。
